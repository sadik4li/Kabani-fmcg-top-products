# 🧼 Kabani FMCG Top Product Analytics Dashboard  

## 📁 Project Overview  

This project presents an interactive **Power BI dashboard** focused on analyzing the performance of the **top five flagship products** within the Kabani FMCG portfolio.  

The dashboard delivers granular insights into **sales volume, profitability, and customer reach** for key products across brands such as **Rinse, Kabani XL, and Rose**.  

By connecting directly to **SQL Server (DirectQuery)**, the solution ensures near real-time data accuracy, enabling stakeholders to monitor KPIs and drill down into detailed product-level performance across multiple locations.

## 📂 Dataset Overview  

The dataset is sourced directly from **SQL Server** and includes:  

- Item-level product descriptions (SKU, pack size, MRP)  
- Financial metrics (Sales, Cost, Profit, Collection)  
- Quantity sold (units/volume)  
- Location codes and regional performance  
- Customer-level sales transactions
